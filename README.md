# About the HEAL Lab
The Healthy Ecologies and Lifestyles Lab is an interdisciplinary health sciences lab at Simon Fraser University studying the social, ecological, and behavioural determinants of health, happiness, and wellbeing. 

# About this repository
This repository holds publicly available datasets, analytic code, and other assets for the Heal Lab projects, including the Canadian Social Connection Survey (CSCS) and the Tips and Recommendations for the Ingestion of Psilocybin (TRIP) survey.
